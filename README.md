
# QrCodeGenerator 🎓

**QrCodeGenerator ** is a professional-grade Student Management System designed for educational institutions. It allows teachers to register student details, generate unique sequential QR codes for ID cards, and maintain a digital record in an Excel-compatible CSV format.

---

## 🚀 Key Features

- **Sequential Data Formatting**: Scanned QR data appears in a clean, line-by-line vertical format.
- **Automated Database**: Every registration is logged with a date stamp into `Student_Database.csv`.
- **Image Vault**: Automatically saves high-quality `.png` QR codes named by Student Roll Number.
- **Optional Field Handling**: Smart logic to handle optional data like Blood Group without breaking the record.
- **Modern UI**: Built with a professional midnight-blue theme and real-time date display.

---

## 🛠️ Tech Stack

* **Language:** Python 3.x
* **GUI Library:** Tkinter
* **Imaging:** Pillow (PIL)
* **QR Engine:** python-qrcode
* **Storage:** CSV (Excel Compatible)
